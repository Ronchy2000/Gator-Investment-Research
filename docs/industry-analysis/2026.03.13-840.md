> OpenClaw：吹响 AIAgent 时代号角

## 核心观点

- OpenClaw 是由彼得・斯坦伯格开发的开源自主人工智能虚拟助理软件项目，历经两次更名，是面向个人与企业的自托管式 AI 数字员工，实现了从 “对话式建议” 到 “自动化执行” 的跨越。
- 需求侧，中国企业 Agent 规模进入加速期，2031 年数量将突破 3.5 亿，年复合增长率超 135%，Token 消耗也将迎来年均超 30 倍的指数级增长；供给侧，GPT-5.4 发布并强化 Agent 能力，大模型持续迭代为 Agent 发展筑牢底层技术基础。
- OpenClaw 的爆发有望推动 AI 产业进入 Agent 时代，让 Agent 成为应用最先落地的场景之一，同时拉动算力需求、提升国产大模型需求并加速其出海。
- 梳理了大模型及应用、算力基础设施两大领域的相关受益标的，并提示了宏观经济、行业竞争等方面的风险。

## 01 OpenClaw：新时代的 AI 员工

### 1.1 OpenClaw 项目背景

OpenClaw 是开源 AI Agent 项目，由软件工程师彼得・斯坦伯格开发，2025 年末以 Clawdbot 为名在 GitHub 发布，因与 Anthropic 公司 Claude 系列产品混淆，2026 年 1 月 27 日更名为 Moltbot，后定名 OpenClaw，2026 年初因能自主处理复杂任务受到关注，发布后 GitHub 星标数快速增长。

### 1.2 OpenClaw 是加强版的 AI Agent

OpenClaw 并非对话式聊天机器人，而是可替代用户执行任务的自主人工智能虚拟助理软件，可部署在 MacOS、Windows 等本地设备，能调用大模型与 API，通过多款即时通讯平台接收指令，完成日程安排、文件整理等工作。其拥有本地优先与隐私可控、强执行能力、多入口无缝接入、模型灵活适配、开源可扩展五大核心特性，还能在本地存储数据与交互历史，具备持久记忆能力。

### 1.3 OpenClaw 允许用户自主化部署

OpenClaw 可完成系统级操作、浏览器自动化、办公自动化等七类真实任务；与 Claude Code、ChatGPT Agents 等同类产品相比，OpenClaw 以本地优先为核心特点，运行在本地硬件，通过聊天应用对话，且开源免费（仅按 API 使用情况计费），目标受众覆盖开发人员与高级用户，而同类产品多为托管代理、闭源收费模式，受众各有侧重。

## 02 需求侧：Agent 需求扩张

### 2.1 AI Agent 是当下应用落地可行方向之一

AI Agent 是能自主感知环境、规划任务、调用工具并执行的 AI 系统，由 “大模型 + 记忆 + 规划 + 工具” 构成，核心差异是具备 “主动性”，区别于传统 AI 助手的 “问答式” 服务，能自主完成全流程任务，既可为个人完成行程规划等事宜，也能在企业端实现发票读取、报销单填写等流程自动化。

### 2.2 国内 Agent 需求有望迎来爆发

据 IDC 测算，受益于本土模型能力升级、智能体技术与应用生态成熟及产业政策支持，中国企业 Agent 规模进入加速期，2031 年数量将突破 3.5 亿，年复合增长率超 135%；同时，任务执行密度和复杂度的提升，将带动智能体 Token 消耗实现年均超 30 倍的指数级跃升，Agent 成为 AI 应用最先落地的场景之一。

## 03 供给侧：大模型持续迭代，底层技术强化

### 3.1 GPT-5.4 正式发布

2026 年 3 月 5 日，openAI 推出 GPT-5.4，是面向专业工作的前沿模型，整合了推理、编程、智能体工作流程的最新技术成果，优化了表格、文档等处理能力，提升了长文本处理能力。该模型提供 Thinking 和 Pro 两个版本，Pro 面向追求复杂任务高性能的用户，Thinking 能生成思考计划、支持任务方向调整，还提升了深度网络研究能力，评测中表现优于此前版本。

### 3.2 GPT 有望在 Agent 能力方面继续强化

GPT-5.4 是 openAI 首个可直接操作计算机的通用模型，能根据截图发出键鼠操作指令，支持智能体跨软件执行复杂工作流，事实准确性也有所改善，且 openAI 将逐步淘汰旧版 GPT-5.2 Thinking 模型。在多项测试中，GPT-5.4 表现亮眼，OSWorld-Verified 测试成功率超人类平均水平，投行基准测试、网络搜索能力测试得分均较前代大幅提升。

### 3.3 大模型能力是 Agent 基石，大模型技术进步推动其商业化落地

AI Agent 是基于大模型能力开发的平台型产品，其当下的爆发，核心原因之一是大模型能力的持续提升。GPT-5.4 强化 Agent 能力、实现原生级 “电脑操作” 功能的技术趋势，表明大模型将 Agent 能力提升作为迭代重点，持续的技术升级将为 OpenClaw 的商业化落地提供底层技术支撑。

## 04 OpenClaw 爆发对产业三大影响

### 4.1 Agent 有望成为应用最先落地场景之一

OpenClaw 是当前用户热度提升最快的产品之一，自推出后 GitHub 星标数快速攀升，爆发后两个月便超越 Linux、React 等发展十多年的超大型基建项目，成为 GitHub 开源软件项目星标数榜首，其热度印证了 Agent 具备成为 AI 应用最先落地场景的潜力。

### 4.2 Token 爆发，云及算力需求有望拉动

OpenClaw 对 Token 消耗的拉动作用显著，在 openrouter 平台中，其 Token 消耗量占比极高，是平台总消耗的主要贡献来源。Token 消耗的加速攀升，进一步带动算力需求增长，维持算力链条通胀趋势；同时，腾讯云智能体开发平台宣布优化模型计费策略，多款公测模型结束免费、混元系列模型涨价，体现了算力与云服务环节的价格调整趋势，与 Agent 需求增长相呼应。

### 4.3 国产化大模型有望崛起

OpenClaw 的 Token 消耗中，国产大模型占比极高，2026 年 3 月 5 日其使用量前五的模型中，阶跃星辰 Step、MiniMax M2.5 两款国产模型合计占总 Token 消耗量约 50%，超过 GPT、Gemini 等海外大模型。同时，国产大模型企业业绩与模型使用量均高速增长，MiniMax 2025 年 Q4 净收入同比大增，2026 年 2 月模型日均 Token 消耗量较去年 12 月增长 6 倍。OpenClaw 的需求激增，让国产大模型凭借能力提升与高性价比优势填补市场需求空白，拉动国产大模型需求提升的同时，推动其实现 Token 出海，并进一步带动国产算力相关需求增长。

## 05 投资分析与风险提示

### 5.1 投资分析

梳理了两大领域的受益标的，涵盖港股、A 股多个上市公司，具体如下：

- 大模型及应用：智谱、MiniMax、科大讯飞、昆仑万维、卓易信息、金山办公、深信服、海天瑞声；
- 算力基础设施：寒武纪、海光信息、华丰科技、华胜天成、云天励飞、优刻得、首都在线、宏景科技、绿联科技、智微智能。

### 5.2 风险提示

- 宏观经济下行风险，可能影响行业整体发展与市场需求；
- 行业竞争加剧，市场参与者增多可能导致行业格局变化；
- 技术开发与应用进度不及预期，大模型、Agent 相关技术研发和落地受阻；
- 需求不及预期，企业与个人对 Agent、大模型相关产品的需求未达测算水平。
