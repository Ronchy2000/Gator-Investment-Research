> “龙虾” 深度：Token “通胀” 谁受益？一 “芯” 二 “模” 三 “云”

## 一、OpenClaw：开启人人可用的 AI Agent 时代

### 1.1 重塑云端的竞争逻辑

OpenClaw 实现 AI 从被动问答工具向 24 小时自主执行、跨平台调度的 “数字员工” 跨越，推动云厂商从算力基础设施方转型为智能体全栈底座服务商，云端竞争核心转向 Agent 部署入口、模型分发与后续扩容。

### 1.2 交互入口、内容社区、API 基建和用户注意力价值的重构

交互入口从分散 App 向 Agent 统一界面收拢，用户行为从信息检索转为意图达成；内容社区成为 Agent 知识获取与协同中枢；API 基建标准化决定 Agent 生态边界；用户注意力价值从流量收割转向任务溢价。

### 1.3 技术与商业收敛的核心是可控性与可结算性

AI 产业关注点从好用的 Agent 转向可衡量、可治理、可收费的层级，计价方式从流量转向单位执行量、安全治理、资源供给计费，核心付费机制包括可执行能力、资源使用、安全治理、授权访问、结果转化付费。

### 1.4 Agent 时代下价值量提升的环节

- **云与 AI 计算平台的重估**：云收入从传统基础设施计费，转向推理、Agent 平台订阅、企业级集成服务、智能体执行行为计费，核心关注推理收入占比、Agent 订阅渗透率等指标。
- **硬件设备从 “被动工具” 向 “主动代理” 的跨越**：存储设备成为 Agent 私有知识库与边缘算力站，物联网设备集成 NPU 实现语义理解与自主响应，硬件向端云协同、高算力方向迭代。
- **安全与治理由可选项转向必需项**：Agent 权限提升带来安全风险，API 安全、Bot 管理、执行审计、异常检测成为核心支出，具备长期合同与高复购特征。
- **事后可追溯链**：企业需要代理执行链路审计，可观测、日志、AI 网关等成为企业级 Agent 标配采购项。
- **搜索与知识平台的商业重构**：内容方从流量广告转向访问授权、付费引用、转化分成、精准答案赞助付费模式。

### 1.5 国内外生态形态导致的付费结构分化

国内大型互联网公司倾向平台闭环发展，云厂商采用整体解决方案捆绑销售，商业化呈现行业方案优先、综合套餐、高合同可见性特征，与海外单一订阅模式存在差异。

### 1.6 付费主体和方向的变化

不同企业部门付费方向明确，CIO 付费云推理与 Agent 平台，安全团队付费安全治理服务，开发运维付费审计可视化，业务线付费优先执行权，市场部门付费答案优先位，政府付费基础设施。

### 1.7 互联网交互单位的改变

Agent 成为互联网主要访问者，交互单位从人类点击转为机器行动与 API 调用，边缘网络、CDN、API 网关、反 Bot 厂商与请求量、策略包绑定，收入确定性提升。

## 二、OpenClaw 安装部署：从体验到深度定制

### 2.1 OpenClaw 核心机制理解

- Workspace：独立智能运行单元，包含记忆与工具，定义行为边界、沉淀长期记忆。
- Skills：标准化模块化能力单元，可开源获取或自主生成，实现 “用对话创造工具”。

### 2.2 OpenClaw 安装部署

分为云托管即用型、大厂云服务器镜像型、国产本地桌面客户端型、国产兼容框架型、原版自托管 + 灵活配置型五类，适配无技术背景用户到专业开发者，成本从免费到弹性付费不等。

## 三、OpenClaw 驱动的三大受益方向

### 3.1 CPU 和国产芯片

- **需求形态蝶变**：Agent 全天候常驻运行，算力需求从单次脉冲转向持续消耗，芯片用量大幅增长。
- **算力结构扩散**：Agent 以 CPU 为控制编排中枢，调度密集型任务激增，CPU 价值重估，或早于 GPU 成为性能瓶颈。
- **需求测算**：全球活跃 Agent 数量、任务执行数、Token 消耗将大幅增长，带动 CPU 需求呈量级提升。
- **国产芯片受益**：算力缺口与自主可控趋势下，寒武纪、海光信息、云天励飞、中国长城、龙芯中科、禾盛新材、天数智芯、壁仞科技等国产算力厂商直接受益。

### 3.2 大模型

- **核心驱动**：OpenClaw 带动 Token 消耗非线性增长，模型单位成本、长文本、多步推理、编程能力成为核心竞争力。
- **MiniMax 优势**：M2.1 模型性价比突出，成本仅为竞品 8%，长文本与编程能力领先，适配高频工作流；多模态能力支撑视觉执行，完成自动化闭环；MoE 架构降低推理成本，编程能力全球领先。
- **智谱优势**：具备全球领先的 coding 技术，模型矩阵覆盖全场景，深度绑定企业协同生态，构建生态闭环与数据飞轮，定价权提升。
- **生态布局**：月之暗面推出 KimiClaw 云端托管方案，智谱推出 AutoClaw 赋能企业办公，MiniMax 绑定绿联 NAS 布局私有云，腾讯推出 Qclaw、WorkBuddy、SkillHub 完善生态。
- **商业化逻辑**：Agent 任务 Token 消耗是传统问答数十倍，带动模型厂营收高增；真实工作流数据构筑壁垒，迁移成本提升，商业模式向高毛利的 Agent-as-a-Service 升级。

### 3.3 云厂商

- **产业演进趋势**：Agent 全天候运行抬升单位用户云消耗，云厂商通过一键部署、价格补贴抢占 Agent 部署入口，锁定用户生态与计费通道。
- **产业链价值分配**：作为 “卖铲人”，云厂商实现三重价值重估，基础实例规模化变现、Token 调用量增长带动弹性收入、AI 平台与工具构筑长期壁垒，从算力分销转向多模型计费层。
- **产品反哺云底座**：模型大厂与终端厂商的开箱即用产品，放大云算力需求；中小厂商成为算力分销商，带动下沉流量与云资源消耗。
- **Token “通胀” 与涨价趋势**：智谱上调模型订阅价格，海外云厂商同步提价，产业定价逻辑从流量转向 Token 消耗，模型厂掌握议价权。
- **云计算产业拐点**：Agent 落地带动云需求提升，国内云厂商走出价格战，毛利率与盈利水平持续改善，云计算迎来收入与利润双增长。

## 四、风险提示

- Agent 落地节奏不及预期，企业适配与采购周期延长影响收入兑现。
- 模型厂产品付费转化率低于预期，用户付费意愿不足制约增长。
- 模型价格战加剧，压缩行业利润空间。
- 算力供给受地缘政治、产能等因素扰动，成本上升压制估值。
- CPU 需求弹性测算存在偏差，技术优化或部署密度不及预期导致需求偏离。
